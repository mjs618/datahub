## 3.  然后将历史存储到平台上

向实时库测点写值，支持批量写入。

请求方式
URL	/api/hsm-db-rtserver/v1/rtdata/node/write
http 请求方式	post
请求参数
序号	名称	类型	是否必填	描述
1	data	Object[]	是	写值对象列表
data[0].nodeId	String	是	测点名，格式为“命名空间:测点名”，如“100:AI001.PV”
data[0].value	Json 基本数据类型	是	值
2	strictMode	Boolean	否	是否采用严格模式，此模式下如果写值对象列表存在核查错误，直接返回失败（code码非0），不会对核查通过的点下发写值命令，默认为false。
响应参数
名称	类型	描述
code	int	返回码，0 成功，其他失败
message	String	返回信息描述
data	int []	每个测点写入结果状态码列表，0 表示成功，其他失败
